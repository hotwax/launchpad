## getting Started

### clone the code
* --> Open the Readme.md file
* --> Open a Terminal window
* --> then clone the launchpad
Run following command to download dependencies
```
npm or npm install
```

### Run the APP
* --> To serve: npm run serve
* --> To build: npm run build
* --> To lint: npm run lint
